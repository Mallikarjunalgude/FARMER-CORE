# FARMER-CORE
# 🌾 KrishiAI – Data-Driven Smart Farming Decision System

🚀 KrishiAI is an AI-powered agricultural assistant that combines **real-world datasets** with **Google AI (Gemini)** to provide farmers with **personalized, actionable decisions**.

---

## 🎯 Problem Statement

Farmers often rely on fragmented information sources such as:

* Weather forecasts
* Soil reports
* Market prices

These disconnected insights lead to:

* Poor crop decisions
* Reduced yield
* Financial losses

---

## 💡 Our Solution

KrishiAI introduces a **Context-Aware Farming Advisory System** that integrates:

* 🌦️ Rainfall & Weather Data
* 🧪 Soil Conditions (pH, NPK)
* 🌱 Crop Requirements
* 🧴 Fertilizer Usage

👉 And transforms them into **real-time, actionable farming decisions**

---

## 🧠 Core Innovation

Unlike traditional apps, KrishiAI:

✔ Combines **multiple agricultural factors**
✔ Compares **real conditions vs ideal crop requirements**
✔ Uses **AI to generate personalized advice**

---

## 📊 Data-Driven Intelligence

We use datasets (Kaggle + agricultural references) to model:

### 🌱 Soil Data

* Soil pH
* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)

### 🌦️ Weather Data

* Rainfall (mm)
* Temperature
* Humidity

### 🌾 Crop Reference Data

* Ideal pH range
* Ideal rainfall
* Recommended fertilizers

---

## ⚙️ How It Works

```text
User Input (Location + Crop + Soil Data)
        ↓
Fetch Weather + Soil + Crop Dataset
        ↓
Compare Actual vs Ideal Conditions
        ↓
Send Context to AI (Google Gemini)
        ↓
Generate Smart Advisory
```

---

## 🤖 AI Output Format

All responses follow a simple, farmer-friendly structure:

1. Problem
2. Solution
3. Prevention

---

## 🌿 Example Output

**Input Conditions:**

* Crop: Tomato
* Soil pH: 5.5 (low)
* Rainfall: 180 mm (high)

**AI Output:**

* **Problem:** Soil is too acidic and excess rainfall
* **Solution:** Add lime to improve pH and reduce irrigation
* **Prevention:** Use balanced fertilizer and monitor rainfall
